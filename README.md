## ✨ Features
- 🔐 Secure admin authentication (JWT)
- 📊 Bulk Excel upload with validation
- 🎓 Auto-generated printable certificates
- 🔍 Public instant verification by Certificate ID
- 📄 One-click PDF download
- 📱 Fully responsive design
- 🌟 Premium Navy & Gold UI with animations

- **Database**: Neon.tech (free MySQL-compatible cloud DB)
